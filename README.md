# Improved Infamy System

A PAYDAY 2 mod that reworks and improves the Infamy system to make progression smoother, more flexible, and less grind-heavy.

## Overview

**Improved Infamy System** focuses on rebalancing the vanilla Infamy mechanics while adding new quality-of-life features. The goal is to reduce unnecessary barriers and give players more control over their Infamy progression.

This mod is actively being worked on, and more upgrades to the system are planned for future updates.

## Features

- **Rebalanced Infamy XP Pool**
  - Infamy XP pool requirement reduced to **5,000,000 XP**

- **Removed Offshore Money Requirement**
  - Offshore cost reduced from **$200,000,000** to **$0**

- **Editable Infamy in Lobby**
  - Infamy level can now be adjusted directly in the lobby

- **Quality-of-Life Improvements**
  - Less grind, faster progression, and more freedom for players

## Installation

1. Make sure you have **SuperBLT** installed.
2. Download or clone this repository.
3. Place the mod folder into your PAYDAY 2 `mods` directory:
4. Launch the game.
